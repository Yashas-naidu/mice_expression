Mice Protein Expression

Predicting the class of the mice based on their protein expression

Guidelines for the input


  1. All inputs must be numeric none of the categorical values are allowed since it is a classification problem and all fields are compulsory
  2. Please enter the genotype expression ratio obtained through the clinical experiment and carefully update all the values
  3. If autofill is used it would take some random assumed value which is only created for testing purpose advised not to consider those predictions which may not be right
  4. Genotype (Control-1,Ts65Dn-0), Treatment (Saline-0,Memantine-1) and Behavior (S/C-1,C/S-0) are categorical values hence 0 or 1 is used because it is one-hot encoded
  5. Similarly the predicted output would be ranging from 0 to 7 which would the classes as follows: c-CS-m (0) c-SC-m (1) c-CS-s (2) c-SC-s (3) t-CS-m (4) t-SC-m (5) t-CS-s (6) t-SC-s (7)
  6. Move to endpoint train to train the model before using it
  7. Check for the dataset from their official website for more details Uc Irvine
